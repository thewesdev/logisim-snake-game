# Logisim Snake Game

Esse é o primeiro projeto da cadeira de Arquitetura de Computadores, o arquivo do projeto pode ser encontrado [aqui](docs/Projeto_01.pdf).

Detalhando o que foi feito nesse projeto, me foi entregue o circuito de um processador didático com algumas instruções escrito no logisim pelo nosso professor, para realizar esse projeto eu só precisei implementar mais duas instruções, BEQ e BNE, das que foram recomendadas pelo professor para corresponder ao código em C que ele disponibilizou para ser transpilado.
